    How to Use SVG Partycle Generator

    This tool converts SVG files into dynamic, interactive particle animations. Here's how to get started:

    Basic Controls

    • Upload SVG: Click "Upload SVG File" or drag and drop your SVG directly into the main area
    • Layer Management: Add multiple SVGs and control their order and visibility from the Layers panel
    • Layer Position: Move and rotate selected SVG layer in 3D space using the position controls
    • SVG Scale: Adjust the overall size of your SVG
    • SVG Depth: Add 3D depth to your particles (z-axis variation) for a more dimensional look
    • Preserve Colors: Enable to keep original SVG colors instead of applying custom colors
    • Include Strokes: Enable to include particles for SVG stroke outlines (like in calligraphy)
    • Stroke Width & Detail: Control how thick and detailed stroke particle outlines appear
    • Particle Count: Control how many particles make up your shape
    • Particle Density: Adjust sampling density (lower values mean more detailed sampling)
    • Size Range: Set minimum and maximum particle sizes for more natural look
    • High Performance Mode: Enable to use instanced rendering for better performance with large numbers of particles
    • Color: Choose between solid color or gradient options (or preserve original SVG colors)
    • Gradient Rotation: Control the direction of the color gradient
    • Animation Speed: Control how quickly particles animate

    Interactive Features

    Layer Management

    Import multiple SVGs and manage them as separate layers. Reorder layers by dragging them up and down, hide/show individual layers, or delete layers you no longer need.

    Layer Positioning

    Select a layer and use the position controls to move and rotate it in 3D space (X, Y, Z). This allows precise positioning of elements.

    Mouse Interaction

    Move your cursor over the particles to see them react. Adjust interaction radius and strength to customize the effect.

    Sand Collider Effect

    Makes particles behave like sand, moving away from your cursor and gradually returning to their original positions.

    Repel Effect

    When enabled, particles move away from your cursor instead of being attracted to it.

    Camera Controls

    Customize orbit sensitivity, zoom and pan speeds to control how the camera responds to your mouse movements.

    Glow Effect & Bloom

    Adds light bloom around particles for a more vibrant appearance. Adjust bloom strength, radius and threshold for precise control.

    Noise Movement

    Adds organic, flowing movement to particles using 3D Perlin noise for more natural animations.

    New Features

    • Layer Management: Import and manage multiple SVGs as separate layers. Control their visibility, order, and delete unwanted layers from the dedicated panel.
    • Layer Positioning: Move and rotate selected layers in 3D space for precise control over your composition.
    • Gradient Rotation: Control the angle and direction of color gradients for more creative possibilities.

    • Original Color Preservation: Keep the original colors from your SVG instead of applying custom colors or gradients.

    Tips & Tricks

    • Layer Management: For complex compositions, build up your scene using multiple SVG layers. Import background elements first, then stack more detailed elements on top.
    • Layer Positioning: Use the position controls to create 3D depth in your scene by placing different elements at different Z positions.
    • Performance: For complex scenes with thousands of particles, enable "High Performance Mode" for better framerates. This uses instanced rendering instead of individual meshes.
    • Preparing SVGs: For best results, use SVGs with clean, simple shapes. Complex SVGs with many elements may require higher particle counts.
    • Working with Strokes: Enable "Include Strokes" with high stroke detail for calligraphy-like effects with particle outlines. Works best with SVGs that have defined stroke paths.
    • Preserving Colors: When working with colorful logos or illustrations, enable "Preserve SVG Colors" to maintain the original design's color scheme.
    • Sand Effect: The sand collider effect creates a more physical interaction, like pushing through sand. Increase return speed for faster particle recovery.
    • Create Depth: Increase SVG Depth for a true 3D effect. Rotate the view with your mouse to see the depth dimension.
    • Fluid Movement: For smoother, more organic animations, enable noise movement and increase the noise scale slightly.
    • Mouse Interaction: For subtle interaction, lower the interaction strength. For dramatic effects, increase both radius and strength.
    • Camera Tips: Reduce orbit sensitivity if rotation feels too fast. Increase zoom speed for quicker zoom in/out actions.
    • Bloom Settings: For a subtle glow, use low bloom strength with high threshold. For dramatic glow, increase strength and radius while lowering threshold.
    • Color Tips: Try different color themes for quick inspiration. Gradients often create more visually interesting results than solid colors.
    • Gradient Rotation: Change the gradient rotation to create different visual effects. Horizontal gradients (0° or 180°) work well for horizon-like designs, while vertical gradients (90° or 270°) work well for height-based effects.
    • Capturing Results: Use the "Screenshot" button for still images, "Export GIF" for animations, or "Export Code" to get code you can use in your own projects.

    Common Issues & Solutions

    • SVG Not Showing: Make sure your SVG has visible elements. Some SVGs may need to be pre-processed to make them compatible.
    • Performance Issues: For complex scenes with thousands of particles, enable "High Performance Mode" which uses instanced rendering for better framerates.
    • Layer Management: If you can't see your newest SVG, check the layer panel to ensure it's visible (eye icon) and properly stacked (might be behind another layer).
    • Colors Not Matching: If your SVG colors don't appear correctly, make sure "Preserve SVG Colors" is enabled in the SVG Source section.
    • Mouse Interaction Not Working: Check that "Mouse Interaction" is enabled. Try increasing the interaction radius and sensitivity.
    • Camera Controls Feel Off: Adjust orbit sensitivity, zoom speed, and pan speed to match your preferences.
    • Particles Looking Too Dense/Sparse: Adjust the particle count and density settings to find the right balance for your SVG.
    • Strokes Not Visible: Increase stroke width and detail values. For some SVGs, you may need to edit the original file to have more defined strokes.
    • Slow Performance: Reduce particle count, disable glow effect, or use simpler SVGs to improve performance. Enable "High Performance Mode" for complex scenes.
    • Sand Effect Too Strong/Weak: Adjust sand strength and return speed to find the right balance for your design.
    • Can't See 3D Effect: Increase the SVG depth value and use your mouse to rotate the view to see the 3D dimension.
    • Bloom Not Visible: Increase bloom strength and radius while decreasing threshold for more noticeable glow effects.
    • GIF Export Taking Too Long: Reduce particle count and animation complexity before exporting GIFs for faster processing.
    • Imported Code Not Working: Make sure to include all required Three.js libraries as listed in the exported code comments.

    About SVG Partycle Generator

    SVG Partycle Generator is a powerful tool that transforms SVG files into mesmerizing interactive particle animations. Built with modern web technologies, it offers designers and developers a simple way to create engaging visual effects.

    The generator uses Three.js for WebGL-powered rendering, allowing for high-performance animations even with thousands of particles. Each SVG is carefully analyzed and converted into particles while preserving its visual form.

    Technical Notes

    This generator works entirely in your browser with no server-side processing. Your SVGs are never uploaded to any server, ensuring your designs remain private and secure.

    High Performance Mode uses Three.js InstancedMesh for dramatically improved rendering efficiency, allowing you to work with 10,000+ particles smoothly even on modest hardware.

    The multi-layer system allows for complex compositions while maintaining good performance through optimized rendering techniques and efficient memory management.

    The generated code is designed to be lightweight and compatible with modern browsers. It requires Three.js and SimplexNoise libraries to function.
